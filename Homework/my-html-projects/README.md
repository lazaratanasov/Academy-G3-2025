# My HTML Project

This project is a simple website consisting of three pages: Home, About Us, and Contact. Each page is linked together through navigation hyperlinks, allowing users to easily navigate the site.

## Project Structure

```
my-html-project
├── index.html      # Home page of the website
├── about.html      # About Us page providing information about the organization
├── contact.html     # Contact page with contact information or a form
└── README.md       # Documentation for the project
```

## Page Descriptions

- **index.html**: This is the home page of the website. It contains links to the About Us and Contact pages, providing an overview of what the site offers.

- **about.html**: This page provides information about the organization or individual. It includes a link back to the home page and a link to the Contact page for further inquiries.

- **contact.html**: This page contains contact information or a contact form. It includes links back to the home page and the About Us page for easy navigation.

## Navigation

Users can navigate through the website using the hyperlinks provided on each page. The structure is designed to be user-friendly and straightforward, ensuring a smooth browsing experience.
